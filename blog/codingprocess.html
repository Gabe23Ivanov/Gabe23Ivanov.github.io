The Process Of Making A Website With HTML, CSS, & JS

This article will give you a very good understanding of the process in building a website.

Posted by Josh Ternyak on September 26, 2019

Introduction

Hold it, stop right there. Do not put all your css into one file. This is one of the biggest mistakes most beginner developers make and they don’t even realize it. I have made this mistake throughout my web development journey and it led me to stress and confusion in my projects. There is a new process of making a website which will get rid of your confusion and stress. Read this article to find out what it is and how to use it in your next web development projects.

What It Is

This new process is called “OCSS” which is an abbreviation for “organized cascasding style sheets.

How To Use It

Step 1: Create a new folder called “mylandingpage”

Step 2: Create an “index.html” file.

Step 3: Create an “generic.css” file, this includes styles for body, *, and any other global styles.

Step 4: Create an “elements.css” file, this includes css that is dedicated to styling html elements.

Step 5: Create an “objects.css” file, this includes css that is dedicated to class-level specifity.

Step 6: Create a “components.css” file, this includes css that is the most important, this is where we begin to style recognizable pieces of UI.

Step 7: Create a “utilities.css” file, this includes css that contains utility and helper classes, hacks and overrides.

Step 8: Begin making your website, and once you get to the css, follow the first 7 steps.

Step 9: Celebrate.

Conclusion
